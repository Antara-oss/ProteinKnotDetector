About the Tool

This pipeline predicts 3D protein structures from raw FASTA sequences and analyzes their mathematical topology to detect complex knots.

How it works:

  1. Structure Prediction: ESMFold predicts the 3D atomic coordinates.
  2. Topology Analysis: The Topoly library computes the Alexander Polynomial using 1,000 stochastic closures.

⚠️ NOTE: For proteins over 400 residues, a sliding window approach is used. This can occasionally result in false negatives if a knotted region falls exactly across a window boundary.
